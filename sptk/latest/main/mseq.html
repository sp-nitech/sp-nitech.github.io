mseq [ option ]

  • -l int

    • output length \((1 \le L)\)

  • -m int

    • output order \((0 \le L - 1)\)

  • stdout

    • double-type m-sequence

The output of this command is \(1\) or \(-1\):

\[ \begin{array}{ccccc} 1, & -1, & 1, & \ldots, & 1 \end{array} \]
where the output length is \(L\). If \(L\) is not given, an inifinite m-sequence is generated.

Parameters

  • [in] argc: Number of arguments.

  • [in] argv: Argument vector.

Return

0 on success, 1 on failure.
